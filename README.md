# visioon_pack 
[![Build Status](https://travis-ci.org/MichalOlejniczak/visioon_pack.svg?branch=master)](https://travis-ci.org/MichalOlejniczak/visioon_pack)

What do you get?

Ubuntu 16.04 container with:
- Python 3.5
- OpenCV 3.1.0 + contrib
- Scikit-learn
- Scikit-image
- Tensorflow 1.4.0 CPU
- Keras 2.1.2
- Jupyter Notebook
- Protobuf
